3DViewer v2.0.

General information

Программа предназначена для визуализации каркасной модели в трехмерном пространстве на языке Си++.

Программа позволяет загружать каркасную модель из файла, перемещать модель, поворачивать модель, масштабировать модель.

Программа имеет графический интерфейс, основанный на Qt.

Интерфейс содержит: кнопку для выбора файла, область визуализации, поля для перемещения, поля для поворота, поле для масштабирования, информацию о загруженной модели.

Программа позволяет задать тип проекции (параллельная и центральная), задать тип ребер, цвет ребер, толщину ребер, методы отображения вершин, цвет вершины, размер вершины.

Также есть возможность выбрать цвет фона.

Кроме того, программа позволяет сохранять изображения в формате JPEG, BMP и записывать GIF-файлы.
